# ROS Workspace for car control

```
packages:
beginner_tutorials  : testing catkin setup
hilens              : communicating with hilens to extract traffic sign info
lanecam             : detect lane lines using external camera
manager             : manage car state
bluetooth_bridge    : vendor code


commands:
setup catkin:
(zsh)  source devel/setup.zsh
(bash) source devel/setup.bash
build:
catkin_make
```
